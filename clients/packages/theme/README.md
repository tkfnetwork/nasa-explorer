# `@ne/theme`

This package contains the core re-useable theme for the frontend applications.

## 🛠️ Tech Stack

Theming uses the following technologies:

- **[tailwind](https://tailwindcss.com/)**

## Usage

To utilise this package simply import the root `index.css` into your tailwind enabled application e.g.

```css
/* css */
@import '@ne/theme/index.css';
```

```ts
/* js/ts */
import '@ne/theme/index.css';
```
